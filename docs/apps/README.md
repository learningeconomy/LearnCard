# Apps

LearnCard powers several user-facing applications. This section documents how users interact with these apps and the key flows they enable.

## Available Apps

### [LearnCard App](learn-card-app/README.md)

The flagship mobile and web wallet for managing verifiable credentials. Users can:

- Claim credentials via links or QR codes
- Store and organize badges, achievements, and IDs
- Share credentials with verifiers
- Connect with other profiles on the LearnCard Network

### [ScoutPass](scouts/README.md)

A specialized app for scouting organizations to issue and manage achievement badges. Features:

- NSO → Troop → Scout hierarchy using Boost parent-child relationships
- BoostID membership cards at each level
- Role-based permissions for badge issuance
- Digital badges for ranks, skills, and achievements
