Custom mount via data-target

By default the widget is auto-placed where the <script> tag appears. Passing data-target="#selector" mounts it inside an existing element — useful when the script must live in <head> or when the layout dictates a specific anchor.

The one line you paste

<div id="claim-here"></div>

<script src="https://cdn.jsdelivr.net/npm/@learncard/embed-sdk@latest/dist/badge-claim.js" async
        data-src="https://learncard.app/demo-badges/accountability-unsigned.json"
        data-target="#claim-here"
        data-mode="both"></script>

Rendered widget

The script tag is at the bottom of <body>, but the widget renders into the #claim-here <div> below.
